PROJECTS

01.

Rock, Paper, Scissors Game

The Rock Paper Scissors game is an interactive, browser-based project built using HTML, CSS, and JavaScript. The game starts by allowing the player to select one of the three choices: Rock, Paper, or Scissors. The computer generates its selection randomly using JavaScript logic. Once both choices are made, the game compares them, evaluates the winner based on the traditional rules (Rock beats Scissors, Scissors beats Paper, Paper beats Rock), and displays the result on the screen.

02.

Calculator

I created a fully functional calculator using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The interface is designed using HTML for structure and CSS for styling, ensuring a clean and user-friendly layout. The calculator's buttons and display are responsive, adapting seamlessly to different screen sizes. JavaScript handles the core functionality, including capturing user input, performing calculations, and dynamically updating the display. It uses event listeners to detect button clicks and applies logic to process the inputs in real-time.
